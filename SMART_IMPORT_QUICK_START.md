## ðŸš¨ Common Issues & Solutions

### Issue: "Preview doesn't match my input"
**Solution:** Check the quick reference - some patterns need specific formatting:
- Use parentheses for fractions: `(a/b)` not `a/b`
- Use `sqrt(x)` not `âˆšx`
- Powers need `^`: `x^2` not `x2`

### Issue: "Greek letters not showing"
**Solution:** Type the full name: `alpha`, `beta`, `theta`, `pi`, etc.

### Issue: "Lost my edits when I closed modal"
**Solution:** This is fixed! But if you see this:
1. Check browser console for errors
2. Ensure JavaScript is enabled
3. Try a different browser

### Issue: "Math looks wrong in saved questions"
**Solution:** 
1. Click "Edit" to review
2. Check the live preview
3. Adjust and save again
